Codecember 2018: Processing

This is the third entry in a recurring project (called Codecember) where every year during the month of December1 I work on something code-related every day3.

In May 2019 I decided to have some fun with Processing and this is what I ended up with. It was cool and fun and I drew inspiration from some weird places4 and some very ordinary places5.

I mainly have three things I wish I would have done differently:

  • I made this using Processing.py because I didn’t want to learn Java. I should have used p5.js, given the fact that this is meant to be displayed on a website.
  • I should have stuck with the circle format. It would have made it a lot more uniform and aesthetic.
  • I should have used more colors. A majority of these are either black or black and red. I should have made this a lot more colorful.

Somewhat unsurprisingly, Codecember 2019 improves on these exact points!
